Properties

  1. object literal - a comma separted list of name/value pairs in {}
  2. define a property by assigning it a value: var myCar = new Object (); myCar.make = "Toyota";
  3. Bracket notation: myCar["make"] = "Toyota";
  4. refer to the value w/o quotes, assign it with quotes
  5. Object literal notation: var myCar = {make: "Toyota", year: 1996}
  6. function displayCar() { var display = "a beautiful " + this.year + " " + this.make; }

Object Values

  1. var flowers = {tulips: "purple", orchids: "white", roses: "blue"}
  2. for ( var i = 0; i < flowers.length; i++ ) {console.log(flowers[i])}
  3. for (flower in flowers) { console.log(flower + " = " + flowers[flower]); }

Objects as sets

  1. var toothpaste = {1: "Crest", 2: "Colgate", 3: "Sensodyne"}
  2. Bracket Notation: delete toothpaste[1];
  3. Dot Notation: delete toothpaste.1;
