Arrays v. Hashes

1-11-2015

Arrays and hashes are collections of objects in ruby. The main difference between them is how they are indexed. In arrays the objects can be called by using an integer: 


        arr = [1, 2, 3]
        arr[0]
        => 1
While elements of a hash are called by a predetermined key provided by the programmer: 
  
        hash = { 
          :food => "taco"
          :drink => "horchata"
          :desert => "flan"
          }
        hash[:dessert]
        => "flan"

Other than that, the main differences are one's of syntax. When creating an array we use brackets: 

 
        []
When creating or denoting a hash we use braces: 
  
        {}
